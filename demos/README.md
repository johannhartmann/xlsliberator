# Serious migration episodes

These eight episodes are public, redistributable migration inputs. They are not
claims that the current release already supports every feature. Each episode
binds a source workbook to a task, observable public acceptance scenarios,
dependencies, restrictions, and exact provenance.

No known-good target is checked in because none has yet been independently
verified against LibreOffice `26.2.4.2` in the pinned Docker runtime. Execution
results belong in evidence bundles and drive generated capability reports.
Hidden acceptance tests are intentionally absent from this repository.

The budget and KPI examples remain Level 0 smoke inputs. These episodes are the
Level 1–3 corpus used for migration planning, repair, security, and behavioral
verification.
